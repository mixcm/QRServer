Mixcm QRServer

Simple and convenient public QR-Code api.

GitHub    联系作者

简单生成

调用方式：https://qrcode.mixcm.cn/generate/?data={data}&height={height}&width={width}&formart={formart}
参数说明：data 为 二维码数据，height 和 width 为高、宽（px），formart 为格式，支持 png、jpg、gif。 调用示例：https://qrcode.mixcm.cn/generate/?data=test&height=200&width=200&formart=jpg

Gravatar

调用方式：https://avatar.mixcm.cn/gravatar/{mailmd5}?s={size}&d={default}&r={r}
调用示例：https://avatar.mixcm.cn/gravatar/605f8c6c64b8fcd514a0b53c6cc3680c?s=100&d=mm&r=X

Github

调用方式：https://avatar.mixcm.cn/github/{username}?s={size}
调用示例：https://avatar.mixcm.cn/github/idawnlight?s=100
( Tip: size 为 0 时即默认大小 )
